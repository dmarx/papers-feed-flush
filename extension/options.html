Paper URL Patterns
Define URL patterns for paper sources. The extension will identify papers based on these patterns.
Each pattern needs:
  • Source ID: A unique identifier for the source (e.g., "arxiv", "nature")
  • Source Name: Display name for the source (e.g., "arXiv", "Nature")
  • URL Pattern: Regular expression to match URLs
  • Paper ID Regex: Regular expression with capture group to extract paper ID
JSON array of pattern objects. Example:
[
  {
    "id": "arxiv",
    "name": "arXiv",
    "urlPattern": "arxiv\\.org\\/(abs|pdf)\\/([0-9]+\\.[0-9]+)",
    "idRegex": "arxiv\\.org\\/(abs|pdf)\\/([0-9]+\\.[0-9]+)"
  },
  {
    "id": "doi",
    "name": "DOI",
    "urlPattern": "doi\\.org\\/([\\w\\.\\-\\/]+)",
    "idRegex": "doi\\.org\\/([\\w\\.\\-\\/]+)"
  }
]
