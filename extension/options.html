Paper URL Patterns
Define URL patterns for paper sources. The extension will identify papers based on these patterns. Each pattern needs:
  • Source ID: A short unique identifier for the source (e.g., arxiv, nature)
  • Source Name: Display name for the source
  • URL Pattern: A regular expression to match against URLs
  • Paper ID Regex: A regex with a capture group to extract the paper ID from the URL
